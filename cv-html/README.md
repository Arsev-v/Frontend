# HTML CV Project Guide

**Project Purpose**  
This project teaches you to build a structured, single-page CV using only HTML. It focuses on semantic layout for education, skills, and career history, while adding SEO meta tags, Open Graph tags for social sharing, and a favicon. This creates a clean foundation ready for CSS styling later.

## Key Requirements

- **Semantic HTML**: Use tags like `<header>`, `<section>`, `<article>`, `<h1>-<h3>`, `<ul>`, `<p>` for proper structure.
- **SEO Meta Tags**: Add `<meta name="description" content="Your CV summary">` and similar essentials in `<head>`.
- **Open Graph (OG) Tags**: Include `<meta property="og:title" content="Your Name CV">`, `og:description`, `og:image`, etc., for social previews.
- **Favicon**: Link it with `<link rel="icon" href="favicon.ico" type="image/x-icon">`.
- **Single-Page Layout**: One HTML file with clear sections; no CSS yet.
